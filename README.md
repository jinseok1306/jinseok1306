## 프론트엔드 개발자 박진석입니다.

## 나의 가치관
1. 타인과의 비교가 아닌 나 자신의 성장에 초점을 맞추자.
2. 실패를 통해서 배우는게 많다. 도전을 두려워하지 말자.
3. 항상 겸손하게, 그리고 자신있게 임하자.
4. 기록하는 습관을 가지자.
5. 지식을 항상 다른 사람들과 공유하여 더 많은 것을 얻자.
